# Requirements

- Web Browser
- VSCode
- `p5.js 2.x Project Generator` Extension
  - https://p5js.org/tutorials/setting-up-your-environment/
- `Live Server` Extension

# Objective

Manipulate variables to create laws of physics within the example: gravity, velocity, boundaries of matter, movement, and relate acceleration (gain of movement) with gravity.

- Control the character using the keyboard arrow keys:
  - `↑` Jump
  - `←` Move left
  - `→` Move right
  - `SHIFT` Run

# Progression Ideas

- Make the environment render itself as the character moves
  - https://p5js.org/examples/repetition-recursive-tree/
- Function to create platforms (and automatically calculate the collision box)
- Use classes
  - https://p5js.org/examples/math-and-physics-smoke-particle-system/
- Use images
  - https://p5js.org/tutorials/animating-with-media-objects/
