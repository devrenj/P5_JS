# Requisitos

- Navegador Web
- VSCode
- Extensão `p5.js 2.x Project Generator`
	- https://p5js.org/tutorials/setting-up-your-environment/
- Extensão `Live Server`
# Objetivo

Manipular variáveis para criar leis da física dentro do exemplo: gravidade, velocidade, limites da matéria, movimento e relacionar ganho de movimento com a gravidade.

- Controlar o personagem com as setas do teclado:
	- `↑` Pular
	- `←` Mover para esquerda
	- `→` Mover para direita
	- `SHIFT` Correr
# Ideias de Progressão

- Fazer o ambiente se desenhar conforme o personagem se movimentar
	- https://p5js.org/examples/repetition-recursive-tree/
- Função para criar plataformas (e calcular o campo de colisão automaticamente)
- Utilizar classes
	- https://p5js.org/examples/math-and-physics-smoke-particle-system/
- Utilizar imagens
	- https://p5js.org/tutorials/animating-with-media-objects/
